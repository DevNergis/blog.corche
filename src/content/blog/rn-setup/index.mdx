## Node.JS 설치

[nodejs 공식사이트](https://nodejs.org/ko)에서 LTS 버전을 다운 후 설치합니다.
![사이트 좌측 하단에 LTS 다운로드 버튼](1.png)이후 터미널에
`node -v` 또는 `npm --version` 을 통해 설치한 버전이 나온다면 정상적으로 설치 된 것입니다.
![정상 설치 확인1](2.png)

<Aside type='tip'>안된다면 터미널 또는 컴퓨터를 한번 껐다 켜주세요!</Aside>

## OpenJDK 설치

다양한 OpenJDK 가 있지만 이 글에서는 MicrosoftJDK 를 설치하겠습니다.<br />(Corretto나 Zulu를 사용해도 되지만 OpenJDK가 설치되는 경로를 알고 있어야 합니다.)

[MicrosoftJDK 공식사이트](https://learn.microsoft.com/ko-kr/java/openjdk/download#openjdk-17)에서 17 버전을 다운 후 설치합니다.
![Windows용 설치 파일 설치](3.png)

이후 터미널에 `java -v` 를 통해 설치한 버전이 나온다면 정상적으로 설치 된 것입니다.
![정상 설치 확인2](4.png)

만일 아래처럼 찾을 수 없다고 나올 경우에는 환경변수를 설정해 줍니다.
![OpenJDK가 정상적으로 설치되지 않은 경우](5.png)

### 환경변수 설정

1. 윈도우 검색에서 "고급 시스템 설정" 을 검색해 "시스템 속성" 의 "고급 탭"을 엽니다.
2. "환경 변수" 를 눌러 환경변수 편집 창을 엽니다.
   ![](6.png)
3. "시스템 변수" 에 "Path" 변수를 찾아 "편집" 버튼을 눌러줍니다.(또는 더블클릭)
   ![](7.png)
4. "새로만들기" 를 누른 후

자신이 선택한 OpenJDK 의 bin 폴더 경로를 입력하고 <kbd>Enter</kbd>를 누르고 "확인" 버튼을 눌러줍니다.(또는 <kbd>Enter</kbd>)

<Aside>
  저는 `C:\Program Files\Microsoft\jdk-17.0.13.11-hotspot\bin` 였으나, 다른 OpenJDK 를 설치한 경우
  파일 경로가 다를 수 있으니, 꼭 자신에게 맞는 bin 폴더 경로를 입력해 주세요.
</Aside>

![](8.png)

5. 이후 터미널에 `java -v` 를 통해 설치한 버전이 나온다면 정상적으로 설치 된 것입니다.
   ![정상 설치 확인2](4.png)

   <Aside type='tip'>안된다면 터미널 또는 컴퓨터를 한번 껐다 켜주세요!</Aside>
